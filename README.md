## Getting Started

clone rep
```bash
 git clone https://github.com/rovan-magdi/Nagwa
```

install npm Package
```bash
 npm i
```

Enter this command to run the react app.
```bash
 npm start
```

Enter this command to run the node app.
```bash
 npm run devStart
```
